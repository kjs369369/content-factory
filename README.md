# 🏭 콘텐츠 팩토리 (Content Factory)

하나의 주제를 입력하면 다양한 플랫폼에 맞는 콘텐츠를 한 번에 생성합니다.

## ✨ 기능

한 번의 입력으로 7가지 콘텐츠 동시 생성:

- 📺 **유튜브 스크립트** - 10-15분 분량, 섹션별 구성
- 📝 **티스토리** - SEO 최적화, HTML 형식
- 📧 **뉴스레터** - 이메일 마케팅용
- 🌿 **네이버 블로그** - 해시태그 포함
- 📰 **칼럼** - 전문적 문체
- 📸 **인스타그램** - 캐러셀 10장 + 캡션 + 해시태그
- 🎬 **숏츠/릴스** - 60초 스크립트

## 🎨 디자인

- 키네틱 타이포그래피 테마
- 블랙 (#000000) + 화이트 (#FFFFFF) + 레드 (#FF0000) + 로열 블루 (#0048BA)
- Bebas Neue 폰트
- 인터랙티브 애니메이션
